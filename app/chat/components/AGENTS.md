# AGENTS

## 目录职责
- 存放聊天页面专用的拆分组件，负责布局与容器结构组织。
- 依据功能拆分顶部栏、消息区、输入区，支持后续能力扩展。

## 子目录结构
- `ChatLayout/`：聊天页面的三段式布局容器，统筹顶部栏、消息区与输入区。
- `ChatLog/`：会话记录滚动视图。
- `Composer/`：聊天输入框与发送按钮组合组件。
- `MessageParts/`：消息片段渲染器注册表，支持多种消息类型（文本、故事卡片等）。

## 关键协作与依赖
- 样式需使用本目录下的 `*.module.scss` 并依赖全局主题变量。
- 存放聊天页面私有的功能组件，包含输入、消息及状态反馈模块。
- 组件需遵循无副作用、可组合的原则，便于在不同聊天布局中复用。
- UI 底层依赖 `antd-mobile` 组件与 `@/styles/variables` 主题变量。
